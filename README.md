
# Twitter Sentiment Analysis 🐦

## CodeAlpha Data Science Internship

### Project Overview
Analyzed **162,000+ tweets** to classify public sentiment as Positive, Negative, or Neutral using Python.

---

### Tasks Completed
| Task | Description |
|------|-------------|
| ✅ Task 2 | Exploratory Data Analysis (EDA) |
| ✅ Task 3 | Data Visualization |
| ✅ Task 4 | Sentiment Analysis using TextBlob |

---

### Dataset
- **Source:** Twitter Data
- **Size:** 162,980 tweets
- **Columns:** `clean_text`, `category`
- **Labels:** -1 (Negative), 0 (Neutral), 1 (Positive)

---

### Libraries Used
```python
pandas | matplotlib | seaborn | textblob
```

---

### Key Findings
- Dataset contains majority **Positive** tweets
- Average tweet length is around **80 characters**
- TextBlob sentiment model achieved reasonable accuracy on the dataset

---

### How to Run
1. Clone this repository
2. Install libraries: `pip install pandas matplotlib seaborn textblob`
3. Open `twitter_analysis.ipynb` in Jupyter or Google Colab
4. Run all cells
